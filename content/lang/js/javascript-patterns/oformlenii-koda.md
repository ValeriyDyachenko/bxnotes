## Горизонтальные отступы

```js
function outer(a, b) {
    var c = 1,
        d = 2,
        inner;
    if (a > b) {
        inner = function () {
            return {
                r: c - d
            };
        };
    } else {
        inner = function () {
            return {
                r: c + d
            };
        };
    }
    return inner;
}
```

## Фигурные скобки

```js
if (true) {
  alert(1);
} else {
  alert(2);
}
```

Фигурная скобка на линии оператора:

```js
function func() {
    return {
        name: "Batman"
    };
}
```
