# Сократить массив

Используйте свойство ```length```, чтобы быстро очисть или сократить массив в JavaScript.

```js
const arr = [11, 22, 33, 44, 55, 66];
arr.length = 3 // [11, 22, 33]
arr.length = 0 // []
```
