# Паттерн один var

Паттерн единственного `var` выглядит следующим образом:

```js
function func() {
 var a = 1,
     b = 2,
     sum = a + b,
     myobject = {},
     i,
     j;
 // код...
}
```

В момент декларирования переменных может совершаться некоторая актуальная работа:

```js
function updateElement() {
 var el = document.getElementById("result"),
     style = el.style;
 // код...
}
```
